Keyboard Shortcuts

Data on key codes can be found at keycode.info

Hit the following keys three times and get a pop up with the corresponding message.

1 Scanning Target / Recipient...
2 Target / Recipient : Fail
3 Target / Recipient : Pass
4 Scanning Target / Sender...
5 Target / Sender : Fail
6 Target / Sender : Pass
7 Inserting into Data Stream...
8 Data Stream Insertion : Fail
9 Data Stream Insertion : Pass
0 Man in the Middle : Complete
